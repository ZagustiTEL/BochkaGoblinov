# BochkaGoblinov

Простой и безопасный веб-мессенджер с шифрованием сообщений и чатом "Избранное".

## 📋 Функциональность

- ✅ Регистрация и авторизация пользователей
- ✅ Поиск и добавление друзей по логину
- ✅ Защищенный чат с шифрованием сообщений
- ✅ Чат "Избранное" для личных заметок
- ✅ Реальное время обновления сообщений
- ✅ Адаптивный интерфейс в стиле WhatsApp
- ✅ Flash-уведомления о событиях

## 🔐 Технологии

- Backend: Python Flask
- Database: SQLite3
- Frontend: HTML5, CSS3, JavaScript
- Шифрование: Fernet (cryptography)
- Безопасность: хеширование паролей (Werkzeug)

## Особенности

Чат "Избранное"

- Автоматически создается для каждого пользователя
- Можно сохранять личные заметки
- Сообщения шифруются как обычные

## Безопасность

- Пароли хранятся в виде хешей
- Сообщения шифруются перед сохранением в БД
- Сессии с защищенным ключом

- Адаптивный дизайн
